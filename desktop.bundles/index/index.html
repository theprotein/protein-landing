Use UI components in graphic editors

Protein is simple service for incredible productivity every day.
It's like Dropbox for your UI components.

Protein sync changes from Sketch and sources in the repositories. All new changes from editors
will be pushed to repository and then Protein will create new pull request
for your front-end developers.

Protein can build vector implementation of your components in the background. Without pain and double work.
It's absolutely fast and intergated with your repositories of sources.

Supported techs for magic

You can create components in your favorite technologies.

Each component can be build with a number of technologies, and every technology
can be replaced: interactions, templates, styles, tests, docs.
Protein support: JSX, BH, BEMHTML as templates; React, ES6 and i-bem.js as interactions;
LESS, Sass and Stylus as preprocessors;
Markdown as docs; Mocha, Jasmine, Karma and Istanbul as test engines;

Also Protein work with Autoprefixer, Autopolyfiller and PostCSS.
