Use UI components in graphical editors

Protein Toolkit is small utility for incredible productivity every day.
It's like Dropbox for your components.

Protein Toolkit sync changes from Sketch, AI and sources updates in the repository. All new changes from editors
will be pushed to repository and then Protein will create new pull request
for your front end developers.

Protein Toolkit can build vector implementation of your components in the background.
Absolutly fast and intergated with your code repositories.

Supported techs for magic

You can create components in your favorite technologies.

Each component can be build with a number of technologies,
each of which can be replaced by a suitable specifically for you: interactions, template, style, test, doc.
Protein support: JSX, BH and SVG as templates; React, ES6 and i-bem.js as interactions; LESS, Sass and Stylus as preprocessors;
Markdown as docs; Mocha, Jasmine, Karma and Istanbul as test engines;

Also Protein work with Autoprefixer, Autopolyfiller and PostCSS.
